# **第一部分：基础理论**

**1. 什么是单纯形法？**

单纯形法（Simplex Method）是求解线性规划问题的一个经典算法。其核心思想很简单：

**在可行域的顶点（角点）之间进行有序搜索，找到使目标函数达到最优值的顶点。**

为什么只需要检查顶点？因为线性规划问题的最优解**必然出现在可行域的某个顶点处**。由于顶点数量是有限的，我们可以通过智能的搜索策略，快速找到最优解。

**3. 为什么从 (0,0) 开始？**

在标准形式的线性规划问题中，起点通常选择 ((x_1, x_2, ..., x_n) = (0, 0, ..., 0))，对应可行域的原点。这是因为：

• 原点通常是可行域的一个顶点

• 容易计算和验证

• 通过引入松弛变量，原点自动成为初始基本可行解

# **第二部分：问题标准化**

**问题陈述**

我们要解决的问题是：

$$\begin{align}

\text{maximize} \quad z &= 3x + 5y \\

\text{s.t.} \quad 2x + 3y &\leq 10 \\

x + 2y &\leq 6 \\

x &\leq 4 \\

y &\leq 3 \\

x, y &\geq 0

\end{align}$$

**转换为标准形式**

线性规划问题必须转换为**等式形式**才能用单纯形法求解。我们通过引入**松弛变量**（slack variables）来实现这一点。

对每个 ($\leq$) 不等式，加入一个松弛变量 ($s_i \geq 0$)：

$$\begin{align}

2x + 3y + s_1 &= 10 \quad \text{（约束1的松弛变量）}\\

x + 2y + s_2 &= 6 \quad \text{（约束2的松弛变量）}\\

x + s_3 &= 4 \quad \text{（约束3的松弛变量）}\\

y + s_4 &= 3 \quad \text{（约束4的松弛变量）}

\end{align}$$

**松弛变量的含义：** ($s_i$) 表示该约束的"剩余容量"。例如，($s_1$) 表示 ($2x + 3y$) 到达上限 10 还需要的"松弛量"。

标准形式的目标函数：

$$z = 3x + 5y + 0s_1 + 0s_2 + 0s_3 + 0s_4$$

# **第三部分：建立初始单纯形表**

**初始可行解**

当 ($x = 0, y = 0$) 时：

• ($s_1 = 10, s_2 = 6, s_3 = 4, s_4 = 3$)

• ($z = 0$)

这就是初始基本可行解。我们称：

• **基变量**（基础变量）：($s_1, s_2, s_3, s_4$)（在基础解中非零）

• **非基变量**：($x, y$)（在基础解中为零）

**初始单纯形表的构造**

单纯形表是一个矩阵形式，将所有约束和目标函数汇总在一起：

**表格说明：**

• 前4行：每一行代表一个约束等式

• 最后一行（z行）：代表目标函数，系数取反号（这样可以统一处理）

• RHS（右端值）：等式右边的常数

# **第四部分：选择进入变量**

**原理：为什么要选最负的系数？**

在最大化问题中，目标函数是 ($z = 3x + 5y$)。

这意味着：

• **增加 ($x$) 一个单位，($z$) 增加 3**

• **增加 ($y$) 一个单位，($z$) 增加 5**

显然，增加 ($y$) 能让 ($z$) 增长得更快！

在单纯形表中，$z$行的系数 (-3) 和 (-5) 代表这个"增长率"：

$$\text{最负的系数} = -5 \quad \Rightarrow \quad z \text{ 增长最快的方向}$$

**最优性条件**

• **当 $z$ 行中所有非基变量的系数都 ($\geq 0$) 时**，已达最优解（无法继续改进）

• **只要还有负系数**，就能继续改进，应该选择最负的系数对应的变量进入基

**我们的例子中**

看初始表的 z 行：

![[Pasted image 20260308185435.png]]

**最负的系数是 -5**，所以 ($\boxed{y \text{ 进入基}}$)

这意味着我们从点 (0, 0) 开始，沿着 ($y$) 轴方向移动，寻找能增加 ($z$) 的下一个顶点。
# 第五部分：选择离开变量（最小比率法）

**原理：为什么需要最小比率法？**

选定了进入变量（($y$) 要增加），但 ($y$) 最多能增加到多少呢？

约束条件限制了 (y) 的增长：

$$\begin{align}

2x + 3y &\leq 10 \quad \Rightarrow \quad y \leq \frac{10}{3} = 3.33 \quad \text{（当 } x=0 \text{ 时）}\\

x + 2y &\leq 6 \quad \Rightarrow \quad y \leq \frac{6}{2} = 3 \quad \text{（当 } x=0 \text{ 时）} \quad \leftarrow \text{最紧}\\

y &\leq 3 \quad \Rightarrow \quad y \leq 3\\

\end{align}$$

**第二个约束最紧**（限制最严格），所以 (y) 最多增加到 3，此时约束2变为等式 (x + 2y = 6)，对应的松弛变量 (s_2 = 0)，即 (s_2) 离开基。

**最小比率法的计算**

在单纯形表中实现这一逻辑的方法是**最小比率法**：

$$\text{比率} = \frac{\text{RHS}}{\text{进入变量列的系数}} \quad \text{（只考虑正系数）}$$

选择**比率最小的行**，该行对应的基变量离开。

**应用到我们的例子**

进入变量是 (y)，计算 (y) 列中每一行的比率：

![[Pasted image 20260308185621.png]]
**最小比率是 3**，对应 ($s_2$) 行，所以 ($\boxed{s_2 \text{ 离开基}}$)

# **第六部分：枢轴操作**

**枢轴元素的定义**

**枢轴元素（pivot element）** 是进入变量列和离开变量行的交叉点处的值。

在我们的例子中：

• 进入变量：($y$)（第2列）

• 离开变量：($s_2$)（第2行）

• **枢轴元素 = 2**（($s_2$) 行的 ($y$) 列值）

  ![[Pasted image 20260308185709.png]]
**行变换（高斯消元）**

目标：通过行变换使 ($y$) 列成为单位向量（只有一个 1，其余为 0）。

**步骤1：** 枢轴行除以枢轴元素

新的 ($s_2$) 行（记为第2行'）：

$$\begin{align}

\text{第2行'} &= \frac{\text{第2行}}{2}\\

&= [1/2, 1, 0, 1/2, 0, 0 | 3]

\end{align}$$

**步骤2：** 使用新的枢轴行消除其他行中的 ($y$) 系数

对 ($s_1$) 行（第1行）：

$$\begin{align}

\text{第1行'} &= \text{第1行} - 3 \times \text{第2行'}\\

&= [2, 3, 1, 0, 0, 0 | 10] - 3 \times [1/2, 1, 0, 1/2, 0, 0 | 3]\\

&= [2 - 3/2, 3-3, 1-0, 0-3/2, 0, 0 | 10-9]\\

&= [1/2, 0, 1, -3/2, 0, 0 | 1]

\end{align}$$

对 ($s_3$) 行（第3行）：系数已为0，不变

对 ($s_4$) 行（第4行）：

$$\begin{align}

\text{第4行'} &= \text{第4行} - 1 \times \text{第2行'}\\

&= [0, 1, 0, 0, 0, 1 | 3] - 1 \times [1/2, 1, 0, 1/2, 0, 0 | 3]\\

&= [-1/2, 0, 0, -1/2, 0, 1 | 0]

\end{align}$$

对 z 行（第5行）：

$$\begin{align}

\text{z行'} &= \text{z行} - (-5) \times \text{第2行'}\\

&= [-3, -5, 0, 0, 0, 0 | 0] + 5 \times [1/2, 1, 0, 1/2, 0, 0 | 3]\\

&= [-3 + 5/2, -5+5, 0, 0+5/2, 0, 0 | 0+15]\\

&= [-1/2, 0, 0, 5/2, 0, 0 | 15]

\end{align}$$

**第九部分：判断最优性**

**最优性检验**

检查 z 行中所有非基变量的系数：

![[Pasted image 20260308190109.png]]
**所有系数都 ($\geq 0$)** ✓

这意味着：

• 如果增加 ($s_1$)（使某个约束松弛），($z$) 会增加 1

• 如果增加 ($s_2$)（使某个约束松弛），($z$) 会增加 1

但 ($s_1$) 和 ($s_2$) 现在都是非基变量（等于0），无法增加。所以**已经无法改进目标函数值**。

**已达到最优解！**

# **总结**

![[Pasted image 20260308190234.png]]


单纯形法通过逐步移动到更优的顶点，最终找到了最优解。这个过程体现了算法的两个核心优势：

1. **有方向性**：每次选择最负的系数，确保沿着最有利的方向改进

2. **有效率**：不需要检查所有顶点，只需迭代几次就找到最优解
